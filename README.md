# Web Template
This is a simple web template that leverages up-to-date frameworks and libs with [React.js](https://facebook.github.io/react) being the main driver on the front end.

## Get Started
### Install
```javascript
npm install
```

### Run
```javascript
gulp
```

### Javascript

* `Relay`
